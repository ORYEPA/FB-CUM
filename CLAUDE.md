# Feliz Cumpleaños — Fernando Banos

Birthday greeting single-page app. React + TypeScript + Vite. Deploy to Vercel.

## Project Context

See `.planning/PROJECT.md` for full project context.
See `.planning/ROADMAP.md` for phases and success criteria.

## Key Facts

- **Name**: "Esclavo Rebelde - Fernando Banos" — fixed in code, not configurable
- **Theme**: Blue / purple / black (not pink/red)
- **No APIs**: No ChatGPT, no Cloudinary — all content is hardcoded/local
- **Photos**: Go in `src/assets/photos/` — add before executing Phase 2
- **Stack**: React + TypeScript + Vite
